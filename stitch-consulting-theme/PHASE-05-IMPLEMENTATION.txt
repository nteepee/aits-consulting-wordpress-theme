================================================================================
TROUBLESHOOTING
================================================================================

Menu not appearing in admin?
  - Check WordPress version (need 6.0+)
  - Verify functions.php is being loaded
  - Check browser console for errors

Mobile toggle not working?
  - Verify assets/js/navigation.js is loaded
  - Check if JavaScript is enabled
  - Test at viewport width < 768px

Submenu not showing?
  - Create nested menu items in WordPress admin
  - Set menu depth to 2 or higher

Dark mode not working?
  - Check OS dark mode setting
  - Verify browser supports prefers-color-scheme
