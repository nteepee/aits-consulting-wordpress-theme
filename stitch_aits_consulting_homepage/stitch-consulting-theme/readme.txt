== Description ==

Stitch Consulting Theme is a professional WordPress Gutenberg block theme designed for AITS Consulting's modern B2B SaaS website. It features a complete design system with custom color palettes, typography hierarchy, responsive breakpoints, and full dark mode support.

Features:
- Modern Gutenberg block editor support
- Dark mode first design
- Complete design system (colors, typography, spacing)
- Inter font family integration
- Responsive breakpoints (sm, md, lg, xl, 2xl)
- Full Site Editing (FSE) support
- Custom block patterns
- Accessibility focused
- Performance optimized

== Installation ==

1. Upload the theme folder to /wp-content/themes/ directory
2. Login to WordPress admin panel
3. Navigate to Appearance > Themes
4. Activate "Stitch Consulting Theme"
5. Go to Appearance > Customize to configure the theme
